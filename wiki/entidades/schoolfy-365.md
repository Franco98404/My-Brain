# Schoolfy 365

**Producto propio SaaS de [[eurekant]]** (no es cliente externo). Plataforma educativa integral para instituciones: comunicación institución–familias + gestión académica.

- **Web:** schoolfy365.com · **App Android:** `com.eureka.schoolfy365`
- **Roles:** Admin, Directivo, Preceptor, Docente, Tutor, Estudiante.

## Estado

En producción y publicado. El doc de producto no se actualiza desde ~nov 2023 (posible stack legacy distinto al actual Flutter/Supabase; hay doc dev con Database + Swagger + API propia). Folder Development con una sola lista (Product Backlog).

## Pipeline comercial

CRM propio (folder "Schoolfy 365" en space CRM): lista **Colegios** (pipeline de instituciones, mayormente en Salta: Colegio del Milagro, San Agustín, Santo Tomás de Aquino, Jean Piaget, etc.) y lista **Contactos**. Etapa de prospección fría.

## Roadmap pendiente
